# EduOS Web App

Next.js 15 frontend for EduOS.

Responsibilities:

- Learner, teacher, parent, admin, and owner interfaces
- Auth-aware routing
- Course and lesson experiences
- Dashboard workflows
- Realtime chat client
- AI Tutor interface

Planned stack:

- Next.js App Router
- TypeScript
- TailwindCSS
- Shadcn UI
- React Hook Form
- TanStack Query
- Zustand
- Socket.io Client
